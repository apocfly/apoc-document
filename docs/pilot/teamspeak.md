## 正式安装

> [!NOTE]
> 本教程以TeamSpeak3-Client-win64-3.6.1为素材制作。

1. 打开"TeamSpeak3-Client-win64-3.6.1.exe"

   ![](./images/teamspeak/1.png)
   ![](./images/teamspeak/2.png)

2. 划到最底下，同意协议

   ![](./images/teamspeak/3.png)

3. 选择安装用户，默认即可

   ![](./images/teamspeak/4.png)

4. 选择安装目录，默认即可

   ![](./images/teamspeak/5.png)
   ![](./images/teamspeak/6.png)
   ![](./images/teamspeak/7.png)
   ![](./images/teamspeak/8.png)

5. 完成安装，弹出以下界面并关闭

   ![](./images/teamspeak/9.png)

---

接下来安装汉化

1. 打开"Chinese_Translation_zh-CN.ts3_translation"

   ![](./images/teamspeak/10.png)

2. 启用汉化

   ![](./images/teamspeak/11.png)

3. 安装成功

   ![](./images/teamspeak/12.png)

## 使用方法

1. 点击 `连接 -> 连接至服务器`

   ![](./images/teamspeak/13.png)

2. 输入以下信息:

   ![](./images/teamspeak/14.png)

    - 服务器别名或地址: ts.apocfly.com
    - 服务器密码: 空
    - 昵称: 填写自己注册的CID

3. 连接成功

   ![](./images/teamspeak/15.png)

4. 进入设置页面

   ![](./images/teamspeak/16.png)

5. 设置PTT并保存

   ![](./images/teamspeak/17.png)
