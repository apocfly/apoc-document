# 航路的查询

## 1. 航路分类

### 公开航路

- A、B、G、R，表示国际（地区）航路航线
- L、M、N、P，表示国际（地区）区域导航航路
- W，表示不涉及周边国家或地区的对外开放航路航线（含 进离场航线）
- Y，表示不涉及周边国家或地区的对外开放区域导航航路V表示对外开放临时航线

### 非公开航路

- P，表示P字点
- H，表示国内航路航线
- Z，表示国内区域导航航路
- J，表示国内进离场航线
- X，表示国内临时航线
- V、X，系列航路需经过ATC许可
